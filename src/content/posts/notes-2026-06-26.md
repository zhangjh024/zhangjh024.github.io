## AI 全栈研发工程师

负责 Web 前端与服务端的全栈研发，承接 AI 视频画布产品（类 LibTV / RunningHub RHTV / TapNow 的无限画布 + 节点式工作流）从架构设计到功能落地。

构建并优化画布交互、节点编排引擎、工作流调度与任务状态管理（多任务并发、长任务轮询、实时进度回传等）。

设计高可用服务端架构，对接 AIGC 模型推理服务与 GPU 算力调度，封装稳定的生成 API 与任务队列。

负责性能优化（前端大画布渲染、WebSocket / SSE 实时通信、服务端高并发）、可观测性与稳定性建设。

与算法、产品紧密协作，将模型能力快速产品化上线。

### 任职要求

本科及以上，计算机相关专业；3 年以上全栈或后端研发经验。

前端：精通 TypeScript / React（或 Vue），有复杂交互 / 画布类项目经验（Canvas、WebGL、节点图编辑器如 React Flow / LiteGraph 等优先）。

服务端：熟练 Node.js / Go / Python 至少一门，熟悉微服务、消息队列、缓存、数据库设计，了解高并发与异步任务架构。

熟悉云原生与部署运维（Docker / K8s、CI/CD），有 GPU 任务调度或推理服务对接经验优先。

有视频画布 / 工作流 / 在线协作类产品研发经验者强烈优先。

加分项：熟悉 ComfyUI 工作流 / 节点协议或做过类似编排引擎；了解 AIGC 模型推理链路（diffusion、视频生成模型部署）；有 0→1 产品从架构到上线的完整经验。
